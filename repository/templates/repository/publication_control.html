Publication


1. Working files retrieval

Download important files from your storage because everything will be deleted after publication.


2. Final documentation and title

Upload the PDF file that is considered as the project's final documentation and finalize project title.


3. Confirmation

Send notification to coordinator for them to check that proper files were uploaded and instructions were strictly followed, and if project title is finalized.

If the coordinator confirmed, the process will proceed to the next step. However, if the coordinator found some anomalies, confirmation will be rejected and the students should resolve the said anomaly.


5. File processing

The uploaded PDF file will be processed and be prepared for publication.


6. DONE!

The project is published.
